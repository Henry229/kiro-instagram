# Product Overview

## Simple Instagram Clone

A social media application that allows users to share photos, interact through likes and comments, and browse personalized feeds. The platform focuses on photo sharing with essential social features including user profiles, following relationships, and chronological content feeds.

## Core Features

- **Photo Sharing**: Upload and share photos with captions
- **Social Interactions**: Like and comment on posts
- **User Profiles**: View user information and post history
- **Follow System**: Follow/unfollow users to curate feed content
- **Personalized Feed**: Chronological display of posts from followed users
- **Authentication**: Secure user registration and login

## Target Experience

The application prioritizes simplicity and core functionality over advanced features. It provides a clean, responsive interface that works across desktop and mobile devices, focusing on essential social media interactions without complexity.
